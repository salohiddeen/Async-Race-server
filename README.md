# Async-Race-server

A simple Node.js JSON server for the Async Race task.

## Features
- In-memory garage and winners data
- Engine start, stop, and drive endpoints
- REST API powered by JSON Server

## Getting started
1. Install dependencies: `npm install`
2. Start the server: `npm start`
3. Open the API at `http://127.0.0.1:3000`

## Scripts
- `npm start` - launch the server
